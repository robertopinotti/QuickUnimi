Come scaricare le video lezioni da Ariel

1. Vai nella pagina di Ariel della videolezione. Premi sul video con il tasto destro. Fai Ispeziona elemento. Copia l'url. Esempio di url: https://videolectures.unimi.it/vod/mp4:ppd-2020-10-27-2.mp4/manifest.m3u8

2. Dall'url che hai copiato, estrai il nome del file compreso tra mp4: e /manifest. Esempio: il nome del file di https://videolectures.unimi.it/vod/mp4:ppd-2020-10-27-2.mp4/manifest.m3u8 è ppd-2020-10-27-2.mp4

Inserisci il nome del file qui dentro e premi invio. Verrai mandato alla pagina del file.
